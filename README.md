Promoter-Classification

Reproduction and improvements for: https://pmc.ncbi.nlm.nih.gov/articles/PMC5291440/

Structure:
- `src/` - code (data loading, models, training)
- `notebooks/` - Colab notebooks and instructions
- `experiments/` - experiment configs and outputs
- `scripts/` - utilities and run scripts
- `PromoterData/` - original FASTA datasets (already in repo)

To run on Colab, open `notebooks/colab_instructions.md`.
